Спочатку потрібно встановити всі бібліотеки pip install fastapi uvicorn pydantic sqlalchemy. Також має бути
встановленим node.js.

Для запуску потрібно запустити термінал та перейти в директорію testovoe, в якій знаходиться файл main.py
після цього треба виконати uvicorn main:app --reload. Має розпочатися завантаження uvicorn сервера.
Після цього переходимо до папки testovoe_vue і виконати npm run dev. В результаті розпочнеться завантаження
серверу. Переходимо за адресою в браузері http://localhost:5173/ де і розміщено сам додаток

З стандартними коефіцієнтами тобто:
pair x1
full house x2
yahtzee x3
three pairs x4
other x0
rtp стає приблизно 128%

З коефіцієнтами:
pair 0.8
full house 1.4
yahtzee 1.9
three pairs 2.7
other 0
Вдається отримати rtp приблизно 95%
