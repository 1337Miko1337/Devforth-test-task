Виконайте git clone https://github.com/1337Miko1337/Devforth-test-task.git
Потім потрібно встановити всі бібліотеки pip install fastapi uvicorn pydantic sqlalchemy. Також має бути
встановленим node.js. На лінукс це виконується за домопомогою sudo apt install nodejs npm

Для запуску потрібно запустити термінал та перейти в директорію Devforth-test-task, в якій знаходиться файл main.py
після цього треба виконати uvicorn main:app --reload. Має розпочатися завантаження uvicorn сервера.
Після цього переходимо до папки testovoe_vue. Відкриваємо ще один термінал. При першому запуску потрібно виконати npm install надалі цього робити буде не потрібно.
Потім виконати npm run dev. В результаті розпочнеться завантаження
серверу. Переходимо за адресою в браузері http://localhost:5173/ де і розміщено сам додаток

З стандартними коефіцієнтами тобто:
pair x1
full house x2
yahtzee x3
three pairs x4
other x0
rtp стає приблизно 111%

З коефіцієнтами:
pair 0.9
full house 1.3
yahtzee 1.7
three pairs 2.6
other 0
Вдається отримати rtp приблизно 95%
